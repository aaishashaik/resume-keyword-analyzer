# Resume Keyword Analyzer

This project analyzes a resume and compares the skills with job description keywords.

It helps job seekers identify missing skills and improve their resumes for better job matching.

## Features
- Extracts keywords from job descriptions
- Compares them with resume skills
- Shows missing or matched skills

## Technologies Used
- Python

## Project Goal
This project is created to help job seekers improve their resumes and understand what skills are required for specific roles.
